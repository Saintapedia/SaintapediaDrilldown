# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.x     | Yes       |

## Reporting a Vulnerability

Please report security vulnerabilities **privately** by emailing
**tom@saintapedia.org** rather than opening a public GitHub issue.

Include:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Your suggested fix, if any

You can expect an acknowledgment within 48 hours. Confirmed vulnerabilities
will be fixed and released within 14 days where possible.
